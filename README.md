## 製品概要
### ブレインストーミング × Tech

### 背景（製品開発のきっかけ、課題等）
- こんかいのプロダクトの開発に至った背景
  - アイデア出しでブレストを行っていた
  - メモを取るために議論が止まってしまう
  - 自分たちは議論に白熱したい
- 着目した顧客・顧客の課題・現状
  - 会議に関わるありとあらゆる人たち
  - ブレインストーミングを効率よくメモする
  - 手動でメモを取るしかない

### 製品説明（具体的な製品の説明）
このアプリはブレインストーミングを自動的にメモします．

#### アプリができること
- 会話から単語を抽出する
- 中心となる単語を表示する
- 単語をリンクする

#### ユーザが行う操作
- 中心となる単語を選択する
- 不必要なリンクを削除する
